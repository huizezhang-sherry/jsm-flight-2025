Boston Marathon (2024)

While the official race results are available on baa.org, location data are not provided. Therefore, a third party website (marathonguide.com) which does have location data was used. Within the US, city/state is provided for each runner; outside the US, only country. Marathonguide.com retrieves this data from runzy.com, but each retrieval is only 20 rows at a time. I used ChatGPT to iteratively query data - with a 2 second timeout between retrievals, since the server was timing out otherwise.
